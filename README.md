# Supply Chain Risk & Sales Analysis 📊

## 📌 Project Overview
[cite_start]تحليل شامل لبيانات سلاسل الإمداد بقيمة **25.3 مليون دولار**[cite: 7, 8]. يهدف المشروع إلى تحديد نقاط الضعف في التوريد وتحليل أداء المبيعات عبر مختلف القطاعات.

## 🚀 Key Insights (النتائج الرئيسية)
* [cite_start]**High-Risk Suppliers:** تم اكتشاف أن **51.4%** من الطلبات تقع ضمن فئة المخاطر العالية[cite: 28, 57].
* [cite_start]**December Bottleneck:** لوحظ ارتفاع متوسط التأخير في شهر ديسمبر ليصل إلى **4.2 يوم**[cite: 133].
* [cite_start]**Top Category:** قطاع الإلكترونيات هو الأعلى مبيعاً بقيمة **5.69 مليون دولار**[cite: 23].
* [cite_start]**Shipping Performance:** الشحن الجوي هو الأسرع، بينما يسجل الشحن البحري والسكك الحديدية أعلى متوسط تأخير[cite: 98, 99, 100].

## 🛠 Tools Used
* **Power BI:** لعمل الداشبورد والتحليل البصري.
* **Supply Chain Metrics:** تحليل الـ Delay Days و Supplier Reliability Score.
